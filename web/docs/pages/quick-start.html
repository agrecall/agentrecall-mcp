Quick Start Guide

Get up and running with AgentRecall in minutes. This guide will walk you through the installation and basic configuration.

Prerequisites

Before you begin, ensure you have the following installed:

  • Docker 20.10+ - Container runtime
  • Docker Compose 2.0+ - Multi-container orchestration
  • Git - Version control (optional, for cloning the repository)

Installation

Option 1: Clone from GitHub

Clone the repository and start the services:

git clone https://github.com/agentrecall/server.git
cd server
cp .env.example .env
docker-compose up -d

Option 2: Download Release

Download the latest release from GitHub:

curl -fsSL -o agentrecall.tar.gz \
  https://github.com/agentrecall/server/releases/latest/download/agentrecall.tar.gz
tar -xzf agentrecall.tar.gz
cd agentrecall
cp .env.example .env
docker-compose up -d

Configuration

Environment Variables

Edit the .env file and set the following required variables:

# Database Configuration
DB_PASSWORD=your_secure_password_here

# JWT Configuration
JWT_SECRET=your_jwt_secret_at_least_32_bytes_long

# OTP Master Key
OTP_MASTER_KEY=your_otp_master_key_at_least_32_bytes_long

Generate Secure Keys

You can generate secure random keys using OpenSSL:

# Generate JWT Secret (32 bytes)
openssl rand -base64 32

# Generate OTP Master Key (32 bytes)
openssl rand -base64 32

Verification

Check Service Health

Verify that all services are running correctly:

# Health check
curl http://localhost:3000/health

# Expected response
{
  "status": "ok",
  "timestamp": "2025-01-15T10:30:00Z",
  "services": {
    "database": "connected",
    "redis": "connected"
  }
}

Test MCP Initialization

Test the MCP protocol initialization:

curl -X POST http://localhost:3000/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "initialize",
    "params": {
      "protocolVersion": "2024-11-05",
      "clientInfo": {
        "name": "test-client",
        "version": "1.0.0"
      }
    }
  }'

Access Admin Panel

Navigate to http://localhost/admin/ to access the admin panel. You can:

  • Register a new account
  • Create API keys
  • View usage statistics
  • Manage users (admin only)
